gravitee illustration

North Walian club promoters Geriant Williams and Joe Harvey, who run a successful event dubbed Gravitee, required the recreation of their logo. This project included redrawing their original 2D logo, producing a 3D equivalent and vectorising both versions for professional reuse at any scale. The project was undertaken by an associate, Charlotte Carbie. Any requests for work that include illustrations will be delegated to Charlotte.
